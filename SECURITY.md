# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by opening an issue at https://github.com/nmhung1210/datapack/issues or emailing the maintainer directly.

We will acknowledge receipt within 48 hours and aim to release a fix within 7 days for critical issues.
